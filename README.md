# 《人工智能数理基础（高级）》

上课地点：上海创智学院，201 教室

上课时间：每周四晚 18:30～21:05

课程内容：聚焦人工智能（artificial intelligence，简称 AI）与科学计算（scientific computing，简称 SC）的交叉研究，一方面通过人工智能技术来解决科学计算问题（AI4SC），另一方面通过科学计算理论来理解人工智能方法（SC4AI）。通过具体案例的学习与实践，启发学生对人工智能与科学计算更深层次的理解与思考。
